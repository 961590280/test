基本数据类型：
	整型 int：存在一个字中（所以长度依赖于机器的类型）。至少16位
	浮点型 float：
	unsigned int（unsingned）:只用于非负数	
函数：参数是值传递（即实参和形参值相同但是互不影响）
	参数
指针：用来存储地址变量
	&地址运算符。可以获取存储地址，打印时使用%p说明符。
	*间接运算符。指向地址中的值。针对后自加操作*pi++,这里发生的操作将会是pi指针的地址自增而不是pi指向的值自增。
	int * p = &q定义变量p为类型为int的指针,储存q变量的地址。p就看做等于&q。
数组：一系列类型相同的数据组成的序列。
	定义数组： int arr[100];
	初始化：int arr[3]={1,2,3};未初始化的数组每个元素的值是不确定的。但是一旦对某个元素复制后会根据数组元素的类型为未赋值的元素赋初值。例如int 的初始值为0，char为a。float为带小数位的0。
		特殊初始化：初始化特定元素 int arr[6]={[5]=10};初始化下标为5的元素的值为10。
		多维数组：可以看做元素为数组的一维数组。层层嵌套。
		指针与数组：数组名也是指针指向数组的头一个元素。给方法传递指针参数时，写数组名就可以了。
				多为数组与指针，int arr[10][10]={...};int(*pi)[10] = arr;如此定义多为数组的指针。即可以看做 指针数组。
